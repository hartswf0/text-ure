CONCEPTS
TRAVERSE
Select source and target media types, then compile a transformation prompt. The output instructs an AI to convert content from source affordances into target structure.
LOOPBACK
When you compile TXT→GAM, the output "becomes" GAM. Send it to a new channel and GAM is auto-selected as source — ready for GAM→IMG or GAM→BOT.
TANGLE
Multiple channels across multiple calculators create a web of transformations. The tangle view shows all connections — branch, send, and track versions.

CAREY Ritual View of Communication

Communication as symbolic world-production, not message transfer.

Cultural Approach Core Axioms
CAREY-CULTURAL-APPROACH-POML-v1
▸
  • A1 Society does not merely survive by communication. Society exists in communication.
  • A2 Communication is not secondary to reality. Communication produces reality.
  • A3 Symbols are simultaneously symbols-of reality and symbols-for reality.
  • A4 All communication models are ethical instruments. They create the worlds they claim to describe.
  • A5 Meaning is not transmitted. Meaning is enacted.

KITTLER Materialist Inscription Systems

Media as technical conditions that determine what can be stored, processed, and retrieved.

Kittler Core Axioms
KITT-URN-GRID-v1
▸
  • A1 Begin with apparatus, not meaning: storage formats determine what a text can become.
  • A2 The grid is address space; movement is a scan-head; the beam is a read operation.
  • A3 Semantics are not primitives; only discrete symbols, allocation, redundancy, routing, and retrieval are primitives.
  • A4 The interface is a surface effect. The mapping is the medium.
  • A5 Noise/glitch is not failure; it is the Real that escapes the symbolic grid.

Spatial Mapping First Principles
FIRST-PRINCIPLES-COMPILE-v1
▸
  • P1 Space is an addressing system, not a stage. Each cell is an addressable register.
  • P2 Two texts: operative text produces surface text. The "meaning" lives in the mapping operation.
  • P3 Geometry is a constraint grammar. Rings = recursion, Spirals = descent, Friezes = time-as-loop.
  • P4 Semantics are functions, not topics. INVOKE | QUESTION | SHOW | LOOP | PIVOT | SEAL | VOID | META.
  • P5 Allocation must be unequal by design. Some lines deserve territory; some share cells.
  • P6 Repetition is architecture. Key phrases become distributed "ribs" that stitch the world.
  • P7 Voids are positive semantic regions. Silence has shape.
  • P8 Reproducibility is required. Given (TextHash, Seed, Version), same traversal → same structure.
